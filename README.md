Шаблон для верстки презентаций с помощью нейросетей. Включает скрипт для автоматической сборки контекста проекта.

Ссылка на репозиторий: https://github.com/sasha2004-art/sample_presentation/

## Алгоритм работы

1. Стиль
Попроси нейросеть подобрать цветовую гамму и шрифты, описав суть и настроение проекта.

2. Ассеты
Сохрани необходимые изображения и логотипы в папку assets/images.

3. Сборка контекста
Запусти скрипт в терминале:
```
python collector.py
```
В корне проекта появится файл code.txt, содержащий структуру и код всех файлов.

4. Генерация
Скопируй содержимое code.txt и отправь в чат с LLM. Добавь промпт с просьбой применить стиль из шага 1 и использовать файлы из папки assets.

5. Итерации
Вставь полученный код в файлы проекта. При необходимости внести правки — повторяй шаги 3 и 4.

## Структура проекта

- collector.py: Скрипт для сборки всего кода в один файл.
- code.txt: Файл с полным контекстом проекта для LLM.
- index.html: Структура и контент слайдов.
- css/style.css: Стили оформления.
- js/main.js: Логика навигации и сохранения прогресса.
- assets/: Папка для изображений и файлов.

## Требования

- Python 3.x (для запуска collector.py)
- Рекомендуется: VS Code + Live Server

## Как сделать форк

Чтобы начать работать со своей копией проекта:

1. Откройте страницу репозитория: https://github.com/sasha2004-art/sample_presentation/
2. Нажмите кнопку Fork в правом верхнем углу страницы.
3. После создания копии нажмите кнопку Code (зеленая) и скопируйте ссылку.
4. Откройте терминал на компьютере и выполните команду:
```
   git clone <ссылка_на_вашу_копию>
```
5. Перейдите в папку проекта:
```
   cd sample_presentation
```
